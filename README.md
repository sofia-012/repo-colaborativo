# Repositorio Colaborativo - Módulos en Python

Descripción del repositorio
Este repositorio fue creado como parte de la actividad "Construcción colaborativa de proyectos en GitHub – Módulos en Python".  
Contiene un programa simple que permite registrar las actividades realizadas durante el día y genera un resumen con el conteo de cada actividad.  
El código está dividido en dos archivos:
- `funcion_2.py` → contiene las dos funciones (módulo)
- `programa_principal.py` → importa el módulo y ejecuta el programa

Cómo usar el programa
1. Abrir `programa_principal.py` en Spyder
2. Ejecutar el archivo
3. Ingresar las actividades del día
4. Escribir "no" o dejar vacío cuando termine**
